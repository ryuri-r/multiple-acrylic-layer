# Multiple Acrylic Layer

투명 PNG 이미지를 아크릴 면으로 쌓아 3D로 확인하는 정적 웹 앱입니다.

## 주요 기능

- 1면으로 시작해 최대 20면까지 추가
- 여러 이미지를 한 번에 선택하면 빈 면을 채우고 필요한 면을 자동 생성
- 빈 면을 먼저 추가한 뒤 각 면에 이미지를 개별 업로드 가능
- PC와 모바일에서 버튼 전체를 눌러 바로 파일 선택 가능
- 면별 PNG, WEBP, JPG 업로드
- 모든 면의 투명 영역을 합친 범위로 아크릴 가로·세로 자동 계산
- 그림 바깥 칼선 여백 2 mm 기본, 0 mm 선택 가능
- 기본 판형은 살짝 둥근 직사각형이며 기본 시점은 평면 정면
- 첫 이미지의 대표색을 연한 파스텔 배경으로 자동 적용
- 면 카드 드래그 순서 변경, 표시/숨김, 삭제
- 미리보기에서 그림 직접 드래그 이동, 그림 더블클릭으로 위치 초기화
- 아크릴 크기, 두께, 면 간격, 분리 보기 조절
- 정면, 입체 시점과 빈 공간 드래그 자유 회전
- 뒤쪽에서 보아도 실제 깊이에 맞는 레이어 순서로 표시
- 현재 3D 화면 PNG 저장
- Y축 좌우 왕복 또는 360° 회전 GIF 저장
- 업로드 이미지는 서버로 전송하지 않고 브라우저 안에서 처리

기본 GIF 모드는 여러 아크릴 면의 깊이 차이를 자연스럽게 보여주는 `Y축 좌우 왕복`입니다.

## 로컬 실행

ES Module을 사용하므로 파일을 직접 더블클릭하지 말고 로컬 서버로 실행합니다.

```powershell
node serve.mjs
```

또는 VS Code Live Server를 사용할 수 있습니다.

## GitHub Pages 배포

1. 이 폴더의 파일을 GitHub 저장소 루트에 업로드합니다.
2. 저장소 `Settings > Pages`로 이동합니다.
3. `Deploy from a branch`를 선택합니다.
4. 배포 브랜치의 `/ (root)`를 선택합니다.

별도 빌드 과정은 필요하지 않습니다.
